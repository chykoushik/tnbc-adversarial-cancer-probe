# tnbc-adversarial-cancer-probe
Adversarial attacks (FGSM + PGD) used as biological discovery tools for Triple Negative Breast Cancer. Not a robustness paper. A biological discovery paper.

The idea is simple. Instead of using adversarial attacks to fool models, we use them to ask every patient the same question — what does the model find most suspicious about your tissue. The gradient maps become biological measurements.

---

## Contributions

1. Adversarial attacks (FGSM + PGD) used as biological discovery tools in TNBC — not for robustness, not for security, but to extract biological signal from model gradients.

2. FGSM and PGD sensitivity analysis applied to TNBC histopathology image classification with biological interpretation of sensitivity maps.

3. Cross-modal adversarial alignment — image sensitivity patterns and gene sensitivity patterns from two independent models tested for biological agreement (Spearman r=0.9636, p=1.85e-06).

4. Two-step BreakHis pretraining competitive with SimCLR pretrained on 2 million cancer patches at a fraction of the compute cost.

5. 11 known TNBC genes recovered in top 20 sensitive genes purely through model stress testing without pathway or gene supervision.

6. Multimodal adversarial fusion model combining image sensitivity and gene sensitivity — AUC 0.9693, shuffled label margin 0.4971.

7. EFNA5 image-gene correlation — image adversarial sensitivity and EFNA5 gene expression significantly correlated (r=−0.71, p=0.015) — a testable biological hypothesis generated by the model.

8. Estrogen response pathway enriched in top sensitive genes (p=1.98×10⁻⁶) — biologically coherent since TNBC is ER negative by definition.

9. Adversarial sensitivity score associated with patient progression-free survival (log-rank p=0.0072, Cox HR=0.54, concordance=0.74).

10. All 11 TNBC patients recovered into a single UMAP cluster using adversarial sensitivity features alone — no labels used.

11. FGSM sensitivity maps capture complementary information to Grad-CAM (mean spatial correlation r=−0.145).

12. Adversarial sensitivity signal generalizes across hospital cohorts — cross-model consistency r=0.95 on external E2 patients.

---

## What This Does

Takes histopathology images and gene expression data from breast cancer patients. Trains image and gene classifiers separately. Then runs FGSM and PGD on both — not to attack the model but to extract per-patient sensitivity maps. Combines image sensitivity and gene sensitivity into a multimodal fusion model.

Best combined model AUC 0.9693 (LOO CV, n=97 patients). Shuffled label margin 0.4971.

---

## Key Findings

Cross-model consistency between ResNet50 and EfficientNet sensitivity maps — Spearman r=0.9636, p=1.85e-06. Two completely different architectures rank the same patients the same way.

TNBC vs non-TNBC adversarial sensitivity is significantly different — Mann-Whitney p=1.45e-06.

11 known TNBC genes recovered in XGB top 20 without being told what to look for.

EFNA5 image-gene correlation r=-0.71, p=0.015. A specific hypothesis the model generated.

Pathway enrichment of top sensitive genes — estrogen response early p=1.98e-06. Biologically correct since TNBC is ER negative by definition.

All 11 TNBC patients land in the same UMAP cluster without labels.

High sensitivity patients have better progression-free survival — log-rank p=0.0072, Cox HR=0.54.

FGSM maps and Grad-CAM maps are weakly negatively correlated (mean r=-0.145). They capture different things.

## Pipeline

```
1. 01_data_loading.ipynb
   Colab. Load A2 and E2 clinical data. Define TNBC labels.
   Merge TCGA gene data with GEO datasets. ComBat batch correction.
   Patient-level train/val/test split. Save processed files to Drive.

2. train_models.py
   Local RTX 4060. ~13 hours.
   Train 5 image classifiers on TCGA-A2 + BreakHis.
   ResNet50 and EfficientNet with Direct, Two-Step, and SSL pretraining.
   Focal loss, WeightedRandomSampler, early stopping, slide-level evaluation.
   Save model weights and training_results.json to Drive.

3. 02_models.ipynb
   Colab. Load trained image models from Drive. Display results and ROC curves.
   Train Random Forest and XGBoost gene classifiers on top 2000 variable genes.
   5-fold stratified CV. Save models, scaler, and gene names to Drive.
   Download METABRIC gene expression via cBioPortal API.
   Apply saved gene classifiers to 155 METABRIC TNBC patients + 155 controls.
   Report external validation AUC.

4. compute_sensitivity_v3.py
   Local RTX 4060. ~40 minutes.
   Compute FGSM and PGD sensitivity maps for all 97 A2 patients (11 TNBC + 86 non-TNBC).
   100 tiles per patient. Both attacks target class 1 TNBC for all patients.
   ResNet50-TS and EfficientNet-TS. Save 6 npy files to Drive.

5. compute_sensitivity_e2.py
   Local RTX 4060. ~20 minutes.
   Same as v3 but for E2 external cohort.
   9 TNBC + 9 matched non-TNBC patients. Save 6 npy files to Drive.

6. 03_adversarial_sensitivity.ipynb
   Colab. Main analysis notebook.
   Load all sensitivity maps. Run statistical tests, biological validation,
   combined model, spatial analysis, Grad-CAM comparison, UMAP clustering,
   pathway enrichment, survival analysis, cross-dataset transfer.
   Save all figures and results to Drive.
```

## Probe Design

Both FGSM and PGD target class 1 (TNBC) for every patient regardless of their true label. This is what makes it a biological probe rather than a standard adversarial attack.

```python
loss = CrossEntropyLoss(output, torch.tensor([1]))  # always TNBC
loss.backward()
sensitivity = img.grad.data.abs()  # what the model finds TNBC-like
```

Every patient gets the same question. The difference in how their tissue responds is the biological signal.
